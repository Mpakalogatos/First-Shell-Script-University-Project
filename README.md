# 📖 First-Shell-Script-University-Project

# 🔭 By working through this project, I have learned these core skills:

  # Handle Command Line Arguments 
  
    The script processes parameters given by the user (like directory name) and inform the user if a required parameter is missing

  # Automate Data Processing and Grading 
  
    Implementing conditional logic to assign grades based on specific results ( Checking if a project result equals to 20 or 10)

  # Manage File System

    The project requires checking for the existence of files and directories, deleting old versions and creating new ones

  # Generate Reports

    The script compiles data from multiple sources (txts and program outputs) into a formatted report (grades.txt) containing names, IDs and calculated total grades

  # Interact with User

    Implemented a prompt that asks the user if they wish to organize files, allowing the scirpt to branch its behavior based on answer
